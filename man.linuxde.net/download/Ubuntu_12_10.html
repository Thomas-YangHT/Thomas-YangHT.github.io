12.10 版Ubuntu更新说明

Ubuntu 12.10 版本，代号为“Quantal Quetzal”，译为“量子绿咬鹃”。采用 Linux 3.5 内核，运行 GNOME 3.6 桌面。

由于该版本的 ISO 文件大小为 800MB，无法用 CD 来刻录，用户只能通过 DVD 或者 USB 设备来进行安装。

该版本相比较其他版本，改进了 Unity 桌面环境，弱化了本地应用和 WEB 应用之间的界限，一些流行的 WEB 应用，如Gmail，则完全可以放置在启动栏中，并将有专属的窗口来运行。

除此之外，该版本改进了 I/O和安全方面，增强了 Unity 桌面体验，整合了 PreViews 预览以及 Amazon 亚马逊搜索，并首次支持 Web 应用程序，同时将 GCC 版本升级为 4.7.2，Python 版本升级为 3.2.3，LibirOffice 办公软件版本更新至 3.6等。
