CentOS 5.10 系统说明

CentOS 5.10是CentOS 5 发行版本系列的第十次更新，它包含了很多错误修正、升级和新功能。

此版本中安装程序被分开为两张 DVD。DVD2中包含的是Open Office所需的其余语言包，如果你不需要安装这些功能，可以只使用DVD1 安装系统。

同样，当使用CD 映像时，只需利用首张 CD 就能安装最小型的 CentOS 系统，但是前提是除了Base（位于 Base System）之外，需要取消选择其它组件。

相比较之前的版本，5.10版本拥有以下几个新特性：
  1. 提供 MySQL 5.1 及 5.5 版。提供 MySQL 5.1 的目的是协助升级 MySQL 5.0 数据库至 MySQL 5.5。请勿把 MySQL 5.1 应用于生产环境中。升级至 MySQL 5.5 之前，建议将数据库数据进行备份。
  2. 由于上述 MySQL 版本的改动，5.0 及 5.1 版的 MySQL 将不会再有安全性更新，若要继续接收 MySQL 的安全性更新，你必须升级至 MySQL 5.5。
  3. 提供 gcc-libraries 这个新组件，内里包含了 libatomic 及 libitm 库。这些库支持某类原子作业及交易性内存。
  4. python-dateutil 及 python-xml 已成为发行版本的一部份，及很可能从 EPEL 中删除。
  5. HP 的 cciss RAID 驱动程序已更新至最新版。
  6. CentOS 5.10 不会再发行 virt-who 组件，CentOS 也不会提供 virt-who 的更新。要是你安装了 virt-who，请尽早利用 'yum remove virt-who' 来删除这个不必要的组件。

与此同时，该版本对自身所包含的组件进行更新和升级，并新增了一些组件。

以上详细信息可查看5.10版本的发行说明。
