OpenStack Ocata User Guides

The central user guides managed by the OpenStack Documentation team were deprecated and removed during the Pike release cycle. The content was moved to the individual project documentation trees . Because of the timing of that change, the Ocata version of the user guides may be out of date. If you encounter problems, please refer either to the version of the guides created for Pike, or the most recent series in development, instead.
