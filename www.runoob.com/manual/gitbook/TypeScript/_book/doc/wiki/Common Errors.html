令人困惑的常见错误

"tsc.exe" exited with error code 1

修复：

  • 检查文件编码，确保为UTF-8 - https://typescript.codeplex.com/workitem/1587

external module XYZ cannot be resolved

修复：

  • 检查模块路径是否大小写敏感 - https://typescript.codeplex.com/workitem/2134
