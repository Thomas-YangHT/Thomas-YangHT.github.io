类 java.util.jar.Manifest
的使用

使用 Manifest 的软件包
java.net 为实现网络应用程序提供类。 
java.util.jar 提供读写 JAR (Java ARchive) 文件格式的类，该格式基于具有可选清单文件的标准 ZIP 文件格式。 
 

java.net 中 Manifest 的使用
 

返回 Manifest 的 java.net 中的方法
 Manifest JarURLConnection.getManifest()
          返回此连接的 Manifest；如果没有，则返回 null。
 

参数类型为 Manifest 的 java.net 中的方法
protected  Package URLClassLoader.definePackage(String name, Manifest man, URL url)
          用该 ClassLoader 中的名称定义一个新包。
 

java.util.jar 中 Manifest 的使用
 

返回 Manifest 的 java.util.jar 中的方法
 Manifest JarInputStream.getManifest()
          返回此 JAR 文件的 Manifest；如果没有，则返回 null。
 Manifest JarFile.getManifest()
          返回 jar 文件清单；如果没有，则返回 null。
 

参数类型为 Manifest 的 java.util.jar 中的构造方法
JarOutputStream(OutputStream out, Manifest man)
          使用指定的 Manifest 创建新的 JarOutputStream。
Manifest(Manifest man)
          构造新的 Manifest，它是指定 Manifest 的副本。
