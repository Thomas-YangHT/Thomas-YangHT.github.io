类 FormattableFlags

字段详细信息

LEFT_JUSTIFY

public static final int LEFT_JUSTIFY
将输出左对齐。根据需要在所转换值的结尾处添加空格 ('\u0020')，以满足字段的最小宽度要求。如果没有设置此标志，则输出将是右对齐的。

此标志对应于格式说明符中的 '-' ('\u002d')。

UPPERCASE

public static final int UPPERCASE
根据创建 formatTo() 方法的 formatter 参数期间给出的 locale 规则，将输出转换为大写形式。输出应该等同于 String.toUpperCase(java.util.Locale) 的以下调用 
     out.toUp2perCase()

此标志对应于格式说明符中的 '^' ('\u005e')。

public static final int ALTERNATE
要求输出使用替换形式。该形式的定义是由 Formattable 指定的。
