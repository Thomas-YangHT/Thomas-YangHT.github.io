public class ActivationGroupID
extends Object
implements Serializable

针对一个注册激活组的标识符，它完成几个目标：

  • 在激活系统中唯一地标识该组，以及
  • 包含指向该组的激活系统的引用，以便该组在必要时能够联络上其激活系统。

ActivationGroupID 由对 ActivationSystem.registerGroup 的调用所返回，并用于标识在该激活系统中的组。当一个激活组被创建/重建时，此组 id 作为激活组的特殊构造方法的参数传入。
