类 java.io.Console
的使用

使用 Console 的软件包
java.io 通过数据流、序列化和文件系统提供系统输入和输出。 
java.lang 提供利用 Java 编程语言进行程序设计的基础类。 
 

java.io 中 Console 的使用
 

返回 Console 的 java.io 中的方法
 Console Console.format(String fmt, Object... args)
          使用指定格式字符串和参数将格式化字符串写入此控制台的输出流中。
 Console Console.printf(String format, Object... args)
          使用指定格式字符串和参数将格式化字符串写入此控制台输出流的便捷方法。
 

java.lang 中 Console 的使用
 

返回 Console 的 java.lang 中的方法
static Console System.console()
          返回与当前 Java 虚拟机关联的唯一 Console 对象（如果有）。
