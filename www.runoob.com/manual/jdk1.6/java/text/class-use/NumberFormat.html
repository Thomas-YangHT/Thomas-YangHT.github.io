类 java.text.NumberFormat
的使用

使用 NumberFormat 的软件包
java.text 提供以与自然语言无关的方式来处理文本、日期、数字和消息的类和接口。 
java.text.spi java.text 包中类的服务提供者类。 
javax.swing.text 提供类 HTMLEditorKit 和创建 HTML 文本编辑器的支持类。

 class DecimalFormat
