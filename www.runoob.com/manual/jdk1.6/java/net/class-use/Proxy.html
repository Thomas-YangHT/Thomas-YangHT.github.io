类 java.net.Proxy
的使用

使用 Proxy 的软件包
java.net 为实现网络应用程序提供类。 
 

java.net 中 Proxy 的使用
 

声明为 Proxy 的 java.net 中的字段
static Proxy Proxy.NO_PROXY
          表示 DIRECT 连接的代理设置，主要通知协议处理程序不使用任何代理。
 

返回变量类型为 Proxy 的类型的 java.net 中的方法
abstract  List<Proxy> ProxySelector.select(URI uri)
          基于访问资源所需的协议和访问资源的目标地址选择所有可用的代理。
 

参数类型为 Proxy 的 java.net 中的方法
 URLConnection URL.openConnection(Proxy proxy)
          与 openConnection() 类似，所不同是连接通过指定的代理建立；不支持代理方式的协议处理程序将忽略该代理参数并建立正常的连接。
protected  URLConnection URLStreamHandler.openConnection(URL u, Proxy p)
          与 openConnection(URL) 相同，不同点在于：通过指定的代理进行连接；不支持代理方式的协议处理程序将忽略该代理参数并进行正常的连接。
 

参数类型为 Proxy 的 java.net 中的构造方法
Socket(Proxy proxy)
          创建一个未连接的套接字并指定代理类型（如果有），该代理不管其他设置如何都应被使用。
