接口 CookiePolicy

public interface CookiePolicy

CookiePolicy 实现决定应该接受哪个 cookie 以及应该拒绝哪个 cookie。提供了三种预定义的策略实现，即 ACCEPT_ALL、ACCEPT_NONE 和 ACCEPT_ORIGINAL_SERVER。

有关更多详细信息，请参阅 RFC 2965 第 3.3 节和第 7 节。

从以下版本开始：
1.6
