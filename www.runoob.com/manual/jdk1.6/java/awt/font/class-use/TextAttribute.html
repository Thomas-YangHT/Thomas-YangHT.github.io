类 java.awt.font.TextAttribute
的使用

使用 TextAttribute 的软件包
java.awt 包含用于创建用户界面和绘制图形图像的所有类。 
java.awt.font 提供与字体相关的类和接口。 
java.awt.im 提供输入方法框架所需的类和接口。 
 

java.awt 中 TextAttribute 的使用
 

返回变量类型为 TextAttribute 的类型的 java.awt 中的方法
 Map<TextAttribute,?> Font.getAttributes()
          返回此 Font 中可用的字体属性的映射。
abstract  Map<TextAttribute,?> Toolkit.mapInputMethodHighlight(InputMethodHighlight highlight)
          返回给定输入方法高亮区的抽象级别描述的可视属性映射，如果不存在映射关系，则返回 null。
 

java.awt.font 中 TextAttribute 的使用
 

声明为 TextAttribute 的 java.awt.font 中的字段
static TextAttribute TextAttribute.BACKGROUND
          用于呈现文本背景的 paint 的属性键。
static TextAttribute TextAttribute.BIDI_EMBEDDING
          文本嵌入级别的属性键。
static TextAttribute TextAttribute.CHAR_REPLACEMENT
          用于取代字符的字体标准字型显示的用户定义字形的属性键。
static TextAttribute TextAttribute.FAMILY
          字体名称的属性键。
static TextAttribute TextAttribute.FONT
          用于提供呈现文本字体的属性键。
static TextAttribute TextAttribute.FOREGROUND
          用于呈现文本的 paint 的属性键。
static TextAttribute TextAttribute.INPUT_METHOD_HIGHLIGHT
          用于输入法高亮显示样式的属性键。
static TextAttribute TextAttribute.INPUT_METHOD_UNDERLINE
          输入法下划线的属性键。
static TextAttribute TextAttribute.JUSTIFICATION
          段落调整的属性键。
static TextAttribute TextAttribute.KERNING
          请求进行字距调整的属性键。
static TextAttribute TextAttribute.LIGATURES
          启用可选连字的属性键。
static TextAttribute TextAttribute.NUMERIC_SHAPING
          将 ASCII 十进制数字转换为其他十进制范围数字的属性键。
static TextAttribute TextAttribute.POSTURE
          字体形态的属性键。
static TextAttribute TextAttribute.RUN_DIRECTION
          线条延伸方向的属性键。
static TextAttribute TextAttribute.SIZE
          字体大小的属性键。
static TextAttribute TextAttribute.STRIKETHROUGH
          删除线的属性键。
static TextAttribute TextAttribute.SUPERSCRIPT
          用于上标和下标的属性键。
static TextAttribute TextAttribute.SWAP_COLORS
          交换前景和背景 Paint 的属性键。
static TextAttribute TextAttribute.TRACKING
          控制跟踪的属性键。
static TextAttribute TextAttribute.TRANSFORM
          字体转换的属性键。
static TextAttribute TextAttribute.UNDERLINE
          下划线的属性键。
static TextAttribute TextAttribute.WEIGHT
          字体 weight 的属性键。
static TextAttribute TextAttribute.WIDTH
          字体宽度的属性键。
 

java.awt.im 中 TextAttribute 的使用
 

返回变量类型为 TextAttribute 的类型的 java.awt.im 中的方法
 Map<TextAttribute,?> InputMethodHighlight.getStyle()
