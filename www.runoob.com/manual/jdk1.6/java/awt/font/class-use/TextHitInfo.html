类 java.awt.font.TextHitInfo
的使用

使用 TextHitInfo 的软件包
java.awt.event 提供处理由 AWT 组件所激发的各类事件的接口和类。 
java.awt.font 提供与字体相关的类和接口。 
java.awt.im 提供输入方法框架所需的类和接口。 
java.awt.im.spi 提供启用可以与 Java 运行时环境一起使用的输入方法开发的接口。
