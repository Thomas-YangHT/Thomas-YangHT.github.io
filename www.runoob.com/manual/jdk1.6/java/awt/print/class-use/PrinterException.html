类 java.awt.print.PrinterException
的使用

使用 PrinterException 的软件包
java.awt.print 为通用的打印 API 提供类和接口。 
javax.swing 提供一组“轻量级”（全部是 Java 语言）组件，尽量让这些组件在所有平台上的工作方式都相同。 
javax.swing.text 提供类 HTMLEditorKit 和创建 HTML 文本编辑器的支持类。 
 

java.awt.print 中 PrinterException 的使用
 

java.awt.print 中 PrinterException 的子类
 class PrinterAbortException
          PrinterAbortException 类是 PrinterException 的子类，用于指示用户或应用程序在打印过程中终止打印作业。
 class PrinterIOException
          PrinterIOException 类是 PrinterException 的子类，用于指示在打印过程中发生了某种 IO 错误。
 

抛出 PrinterException 的 java.awt.print 中的方法
abstract  void PrinterJob.print()
          打印一组页面。
 int Printable.print(Graphics graphics, PageFormat pageFormat, int pageIndex)
          将指定索引处的页面用指定格式打印到指定的 Graphics 上下文。
 void PrinterJob.print(PrintRequestAttributeSet attributes)
          使用属性集中的设置打印一组页面。
 void PrinterJob.setPrintService(PrintService service)
          将此 PrinterJob 与新的 PrintService 关联。
