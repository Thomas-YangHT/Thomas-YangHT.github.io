类 java.awt.MenuComponent
的使用

使用 MenuComponent 的软件包
java.awt 包含用于创建用户界面和绘制图形图像的所有类。 
 

java.awt 中 MenuComponent 的使用
 

java.awt 中 MenuComponent 的子类
 class CheckboxMenuItem
          此类表示一个可包括在菜单中的复选框。
 class Menu
          Menu 对象是从菜单栏部署的下拉式菜单组件。
 class MenuBar
          MenuBar 类封装绑定到框架的菜单栏的平台概念。
 class MenuItem
          菜单中的所有项必须属于类 MenuItem 或其子类之一。
 class PopupMenu
          此类实现能够在组件中的指定位置上动态弹出的菜单。
 

参数类型为 MenuComponent 的 java.awt 中的方法
 void MenuBar.remove(MenuComponent m)
          从此菜单栏移除指定的菜单组件。
 void Menu.remove(MenuComponent item)
          从此菜单移除指定的菜单项。
 void Frame.remove(MenuComponent m)
          从此窗体移除指定的菜单栏。
 void Component.remove(MenuComponent popup)
          从组件移除指定的弹出菜单。
 void MenuContainer.remove(MenuComponent comp)
