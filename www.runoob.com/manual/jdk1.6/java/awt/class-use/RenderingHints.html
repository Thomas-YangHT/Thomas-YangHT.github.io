类 java.awt.RenderingHints
的使用

使用 RenderingHints 的软件包
java.awt 包含用于创建用户界面和绘制图形图像的所有类。 
java.awt.image 提供创建和修改图像的各种类。 
java.awt.image.renderable 提供用于生成与呈现无关的图像的类和接口。
