java.security.spec
类 ECPoint

java.lang.Object
  java.security.spec.ECPoint

public class ECPoint
extends Object

此不可变类在仿射坐标中表示椭圆曲线 (EC) 上的点。其他坐标系可以扩展此类，以便在其他坐标中表示此点。

1.5
