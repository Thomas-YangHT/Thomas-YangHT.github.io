类 java.security.spec.ECPoint
的使用

使用 ECPoint 的软件包
java.security.interfaces 提供的接口用于生成 RSA Laboratory Technical Note PKCS#1 中定义的 RSA（Rivest、Shamir 和 Adleman AsymmetricCipher 算法）密钥，以及 NIST 的 FIPS-186 中定义的 DSA（数字签名算法）密钥。 
java.security.spec 提供密钥规范和算法参数规范的类和接口。 
 

java.security.interfaces 中 ECPoint 的使用
 

返回 ECPoint 的 java.security.interfaces 中的方法
 ECPoint ECPublicKey.getW()
          返回公用点 W。
 

java.security.spec 中 ECPoint 的使用
 

声明为 ECPoint 的 java.security.spec 中的字段
static ECPoint ECPoint.POINT_INFINITY
          这就将该点定义为无穷大。
 

返回 ECPoint 的 java.security.spec 中的方法
 ECPoint ECParameterSpec.getGenerator()
          返回也称为基点的生成器。
 ECPoint ECPublicKeySpec.getW()
          返回公用点 W。
 

参数类型为 ECPoint 的 java.security.spec 中的构造方法
ECParameterSpec(EllipticCurve curve, ECPoint g, BigInteger n, int h)
          创建基于指定值的椭圆曲线域参数。
ECPublicKeySpec(ECPoint w, ECParameterSpec params)
          根据指定参数值，创建一个新的 ECPublicKeySpec。
