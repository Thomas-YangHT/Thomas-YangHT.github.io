的使用

使用 KeyStore 的软件包
java.security 为安全框架提供类和接口。 
java.security.cert 提供用于解析和管理证书、证书撤消列表 (CRL) 和证书路径的类和接口。 
javax.net.ssl 提供用于安全套接字包的类。
