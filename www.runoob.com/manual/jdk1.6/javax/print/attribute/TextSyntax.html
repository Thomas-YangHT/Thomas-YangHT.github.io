public abstract class TextSyntax
extends Object
implements Serializable, Cloneable

类 TextSyntax 是一种抽象的基类，它提供其值是字符串的所有属性的公共实现。文本属性包括指示自然语言的语言环境。因此，文本属性总是表示本地化字符串。一旦构造之后，文本属性的值就是不可变的。

TextSyntax

protected TextSyntax(String value,
                     Locale locale)
使用指定的字符串和语言环境构造 TextAttribute。

参数：
value - 文本字符串。
locale - 文本字符串的自然语言。将 null 解释为默认的语言环境，这是 Locale.getDefault() 返回的语言环语。
抛出：
NullPointerException - （未经检查的异常）如果 value 为 null，则抛出该异常。

equals

public boolean equals(Object object)
返回此文本属性是否等效于传入的对象。若要等效，以下所有条件都必须为 true：
  1. object 不为 null。
  2. object 是类 TextSyntax 的一个实例。
  3. 此文本属性的底层字符串和 object 的底层字符串相等。
  4. 此文本属性的语言环境和 object 的语言环境相等。
