使用 DocAttributeSet 的软件包
javax.print 为 JavaTM Print Service API 提供了主要类和接口。 
javax.print.attribute 提供了描述 JavaTM Print Service 属性的类型以及如何分类这些属性的类和接口。 
 

javax.print 中 DocAttributeSet 的使用

 DocAttributeSet SimpleDoc.getAttributes()

参数类型为 DocAttributeSet 的 javax.print 中的构造方法
