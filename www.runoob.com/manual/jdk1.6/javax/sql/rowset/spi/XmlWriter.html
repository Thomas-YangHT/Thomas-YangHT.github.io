接口 XmlWriter

RowSetWriter
public interface XmlWriter
extends RowSetWriter

一个专用接口，用来方便地扩展 SyncProvider 抽象类以用于面向 XML 的同步提供者。

提供 XML 数据 writer 功能（如输出 XML 流功能）的 SyncProvider 实现可以实现此接口，以向 WebRowSet 实现提供标准 XmlWriter 对象。

编写 WebRowSet 对象包括打印 rowset 的数据、元数据和属性，都使用适当的 XML 标记。

方法详细信息

writeXML

void writeXML(WebRowSet caller,
              Writer writer)
              throws SQLException
将给定的 WebRowSet 对象以 XML 文档的形式编写到指定的 java.io.Writer 输出流。此文档包括 rowset 的数据、元数据、属性，再加上适当的 XML 标记。

caller 参数必须是 WebRowSet 对象，该对象的 XmlWriter 字段包含对此 XmlWriter 对象的引用。

参数：
caller - 要编写的 WebRowSet 实例，此 XmlWriter 对象是它的 writer
writer - java.io.Writer 对象，它充当输出流，用于以 XML 文档形式编写 caller
抛出：
SQLException - 如果发生数据库访问错误或此 XmlWriter 对象不是给定 WebRowSet 对象的 writer
