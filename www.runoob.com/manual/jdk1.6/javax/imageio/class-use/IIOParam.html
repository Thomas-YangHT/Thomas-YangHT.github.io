类 javax.imageio.IIOParam
的使用

使用 IIOParam 的软件包
javax.imageio Java Image I/O API 的主要包。 
javax.imageio.plugins.bmp 包含供内置 BMP 插件使用的公共类的包。 
javax.imageio.plugins.jpeg 支持内置 JPEG 插件的类。 
 

javax.imageio 中 IIOParam 的使用
 

javax.imageio 中 IIOParam 的子类
 class ImageReadParam
          描述如何对流进行解码的类。
 class ImageWriteParam
          描述如何对流进行编码的类。
 

参数类型为 IIOParam 的 javax.imageio 中的方法
 boolean IIOParamController.activate(IIOParam param)
          激活控制器。
 

javax.imageio.plugins.bmp 中 IIOParam 的使用
 

javax.imageio.plugins.bmp 中 IIOParam 的子类
 class BMPImageWriteParam
          用于以 BMP 格式编码图像的 ImageWriteParam 的子类。
 

javax.imageio.plugins.jpeg 中 IIOParam 的使用
 

javax.imageio.plugins.jpeg 中 IIOParam 的子类
 class JPEGImageReadParam
          此类添加了使用内置 JPEG reader 插件时设置 JPEG 量化表和 Huffman 表的能力。
 class JPEGImageWriteParam
          此类添加了使用内置 JPEG writer 插件时设置 JPEG 量化表和 Huffman 表以及请求为图像计算最佳 Huffman 表的能力。
