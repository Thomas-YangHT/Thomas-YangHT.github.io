软件包 javax.lang.model.type
的使用

使用 javax.lang.model.type 的软件包
javax.lang.model.element 用于 Java 编程语言的模型元素的接口。 
javax.lang.model.type 用来为 Java 编程语言类型建立模型的接口。 
javax.lang.model.util 用来帮助处理程序元素和类型的实用工具。
