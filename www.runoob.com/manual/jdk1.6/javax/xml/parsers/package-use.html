软件包 javax.xml.parsers
的使用

使用 javax.xml.parsers 的软件包
javax.xml.bind.annotation 定义将 Java 程序元素定制成 XML 模式映射的注释。 
javax.xml.parsers 提供允许处理 XML 文档的类。 
 

javax.xml.bind.annotation 使用的 javax.xml.parsers 中的类
DocumentBuilder
          定义 API， 使其从 XML 文档获取 DOM 文档实例。
 

javax.xml.parsers 使用的 javax.xml.parsers 中的类
DocumentBuilder
          定义 API， 使其从 XML 文档获取 DOM 文档实例。
DocumentBuilderFactory
          定义工厂 API，使应用程序能够从 XML 文档获取生成 DOM 对象树的解析器。
ParserConfigurationException
          指示一个严重的配置错误。
SAXParser
          定义包装 XMLReader 实现类的 API。
SAXParserFactory
          定义工厂 API，使应用程序能够配置和获取基于 SAX 的解析器以解析 XML 文档。
