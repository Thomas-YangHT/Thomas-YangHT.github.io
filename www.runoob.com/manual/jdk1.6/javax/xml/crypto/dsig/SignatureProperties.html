javax.xml.crypto.dsig
接口 SignatureProperties

所有超级接口：
XMLStructure
public interface SignatureProperties
extends XMLStructure

W3C Recommendation for XML-Signature Syntax and Processing 中定义的 XML SignatureProperties 元素的表示形式。XML Schema Definition 的定义如下： 


<element name="SignatureProperties" type="ds:SignaturePropertiesType"/> 
<complexType name="SignaturePropertiesType">
<sequence>
<element ref="ds:SignatureProperty" maxOccurs="unbounded"/> 
</sequence>
<attribute name="Id" type="ID" use="optional"/> 
</complexType>
调用 XMLSignatureFactory 类的 newSignatureProperties 方法可以创建 SignatureProperties 实例，例如： 
XMLSignatureFactory factory = XMLSignatureFactory.getInstance("DOM");
SignatureProperties properties = 
        factory.newSignatureProperties(props, "signature-properties-1");

从以下版本开始：
1.6
另请参见：
