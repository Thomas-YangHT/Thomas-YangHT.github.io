使用 Data 的软件包
javax.xml.crypto 用于 XML 加密的通用类。 
javax.xml.crypto.dsig 用于生成和验证 XML 数字签名的类。 
javax.xml.crypto.dsig.keyinfo 用来解析和处理 KeyInfo 元素和结构的类。
