软件包 javax.xml.stream

接口摘要
EventFilter 此接口声明了一个简单的过滤器接口，用户可创建它来过滤 XMLEventReader。
Location 提供有关事件位置的信息。
StreamFilter 此接口声明了一个简单的过滤器接口，用户可创建它来过滤 XMLStreamReader。
XMLEventReader 此类是用于解析 XML 事件的顶层接口。
XMLEventWriter 此类是用于编写 XML 文档的顶层接口。
XMLReporter 此接口用于报告非严重错误。
XMLResolver 此接口用于在 XML 解析过程中解析资源。
XMLStreamConstants 此接口声明此 API 中所使用的常量。
XMLStreamReader XMLStreamReader 接口允许转发 XML 和对 XML 的只读访问。
XMLStreamWriter XMLStreamWriter 接口指定如何编写 XML。
 

类摘要
XMLEventFactory 此接口定义一个创建 XMLEvent 实例的实用工具类。
XMLInputFactory 定义用于获取流的工厂的抽象实现。
XMLOutputFactory 定义用于获取 XMLEventWriter 和 XMLStreamWriter 的工厂抽象实现。
 

异常摘要
XMLStreamException 用于不可预料处理错误的基本异常。
 

错误摘要
FactoryConfigurationError 一个用于报告工厂配置错误的错误类。
