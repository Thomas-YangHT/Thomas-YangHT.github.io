软件包 javax.xml.bind.annotation.adapters

XmlAdapter 及其规范定义的子类允许任意 Java 类与 JAXB 一起使用。

请参见：
          描述

类摘要
CollapsedStringAdapter 用来处理 xs:token 及其派生类型的内置 XmlAdapter。
HexBinaryAdapter xs:hexBinary 的 XmlAdapter。
NormalizedStringAdapter 处理 xs:normalizedString 的 XmlAdapter。
XmlAdapter<ValueType,BoundType> 修改 Java 类型以适应自定义编组。
XmlJavaTypeAdapter.DEFAULT 此类在 XmlJavaTypeAdapter.type() 中使用，用来表示从字段、属性、参数或类的签名推导出的类型。
 

注释类型摘要
XmlJavaTypeAdapter 对自定义编组使用实现 XmlAdapter 的适配器。
XmlJavaTypeAdapters 多个 @XmlJavaTypeAdapter 注释的容器。
