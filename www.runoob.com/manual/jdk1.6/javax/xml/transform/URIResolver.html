javax.xml.transform
接口 URIResolver

public interface URIResolver

实现此接口的对象可由处理器进行调用，以将 document()、xsl:import 或 xsl:include 中使用的 URI 转换为 Source 对象。

方法摘要
 Source resolve(String href, String base)
          当遇到 xsl:include、xsl:import 或 document() 函数时，由处理器调用此方法。
 

方法详细信息

resolve

Source resolve(String href,
               String base)
               throws TransformerException
当遇到 xsl:include、xsl:import 或 document() 函数时，由处理器调用此方法。

参数：
href - href 属性，可以是相对的或绝对的。
base - 首个参数将相对于其成为绝对参数的基本 URI（如果需要绝对 URI）。
返回：
Source 对象，如果不能解析 href，则返回 null，且处理器本身应尝试解析 URI。
抛出：
TransformerException - 如果尝试解析 URI 时发生错误。
