软件包 javax.swing.plaf 的描述

提供一个接口和许多抽象类，Swing 用它们来提供自己的可插入外观功能。它的类由外观 UI（比如 Basic 外观和 Java 外观 (Metal)）实现并为其创建子类。此包仅由那些无法通过为现有外观组件（比如 javax.swing.plaf.basic 和 javax.swing.plaf.metal 包提供的那些组件）创建子类来创建新外观的开发人员使用。

注：大多数 Swing API 都不是 线程安全的。有关细节，请参阅 The Java Tutorial 中的 Threads and Swing 一节。
