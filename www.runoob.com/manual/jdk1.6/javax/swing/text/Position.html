javax.swing.text
接口 Position

public interface Position

表示文档中的位置。该接口能提取出文档的实现详细信息，并启用文档内能够随编辑文档跟踪更改的位置规范。

Position 对象指向两个字符之间的位置。周围内容改变时，Position 对象自动调整其偏移量以反映更改。如果在 Position 对象的位置之前插入或移除内容，则 Position 分别增加或减少其偏移量以便指向相同位置。如果文档中包含 Position 偏移量的部分被移除，则该 Position 的偏移量就成为移除区域的开始位置。例如，如果 Position 的偏移量为 5 而 2-10 的区域被移除，那么该 Position 的偏移量就成为 2。

偏移量为 0 的 Position 是一种特殊情况。文档内容改变时它不改变其偏移量。
