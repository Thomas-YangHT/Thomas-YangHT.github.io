javax.swing.event
接口 DocumentEvent.ElementChange

所有已知实现类：
AbstractDocument.ElementEdit
正在封闭接口：
DocumentEvent
public static interface DocumentEvent.ElementChange

描述对特定元素所做的更改。

方法摘要
 Element[] getChildrenAdded()
          获得添加到给定父元素中的子元素。
 Element[] getChildrenRemoved()
          获得从给定父元素中移除的子元素。
 Element getElement()
          返回所表示的元素。
 int getIndex()
          获得所表示元素中的索引。
 

方法详细信息

getElement

Element getElement()
返回所表示的元素。此元素是已被更改的元素。

返回：
元素

getIndex

int getIndex()
获得所表示元素中的索引。此索引是子元素被添加和/或移除的位置。

返回：
大于或等于 0 的索引

getChildrenRemoved

Element[] getChildrenRemoved()
获得从给定父元素中移除的子元素。所返回的元素数组是按照元素在文档中的放置顺序存储的，并且必须彼此相邻。

返回：
子元素

getChildrenAdded

Element[] getChildrenAdded()
获得添加到给定父元素中的子元素。所返回的元素数组按元素在文档中的放置顺序，并且必须彼此相邻。

返回：
子元素
