javax.management.remote
接口 JMXConnectorServerProvider

public interface JMXConnectorServerProvider

使用给定的协议创建 JMX API 连接器服务器的提供者。此接口的实例由 JMXConnectorServerFactory 作为其 newJMXConnectorServer 方法的一部分创建。

从以下版本开始：
1.5

方法摘要
 JMXConnectorServer newJMXConnectorServer(JMXServiceURL serviceURL, Map<String,?> environment, MBeanServer mbeanServer)
          创建位于给定地址的新连接器服务器。
 

方法详细信息

newJMXConnectorServer

JMXConnectorServer newJMXConnectorServer(JMXServiceURL serviceURL,
                                         Map<String,?> environment,
                                         MBeanServer mbeanServer)
                                         throws IOException

创建位于给定地址的新连接器服务器。每个对此方法的成功调用生成不同的 JMXConnectorServer 对象。

参数：
serviceURL - 新的连接器服务器的地址。新的连接器服务器的 getAddress 方法返回的连接器服务器的实际地址不一定完全相同。例如，它可能包含一个端口号，而原始地址中并没有该端口号。
environment - 包含用于控制新连接器服务器行为的命名属性的只读 Map。此映射中关键字必须为字符串。每个关联值的适合类型取决于属性。
mbeanServer - 该连接器服务器连接到的 MBean 服务器。如果该连接器服务器将通过注册的方式连接到 MBean 服务器，则该参数为 Null。
返回：
代表新的连接器服务器的 JMXConnectorServer。每次成功调用此方法都生成不同的对象。
抛出：
NullPointerException - 如果 serviceURL 或 environment 为 null。
IOException - 如果连接器服务器无法创建。
