VAR

名称

VAR -- 定义一个变量

大纲

VAR varname IS ctype

描述

VAR命令将新的C数据类型分配给宿主变量。 宿主变量必须预先在声明段声明。

参数

varname

C变量名字。

ctype

C类型说明。

例子

Exec sql begin declare section;
short a;
exec sql end declare section;
EXEC SQL VAR a IS int;

兼容性

VAR命令是PostgreSQL扩展。
