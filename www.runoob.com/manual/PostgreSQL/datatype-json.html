8.14. JSON 类型

json数据类型可以用来存储JSON（JavaScript Object Notation）数据， 就像RFC 4627中指定的那样。 这样的数据也可以存储为text，但是json 数据类型更有利于检查每个存储的数值是可用的JSON值。 这里也有相关的支持函数可用；参阅第 9.15 节。

PostgreSQL只允许每个数据库用一种服务器编码。因此， 使JSON严格符合规范是不可能的，除非服务器编码是UTF-8。 试图直接包含不能在服务器编码中表示的字符将会失败；相反的， 能在服务器编码中表示但是不在UTF-8中的字符是被允许的。\uXXXX 转义是被允许的，无视服务器编码，并且只检查语法的正确性。
