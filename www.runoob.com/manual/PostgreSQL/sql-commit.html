COMMIT

名称

COMMIT -- 提交当前事务

大纲

COMMIT [ WORK | TRANSACTION ]

描述

COMMIT提交当前事务。 所有事务的更改都将被其它事务可见， 而且保证即使有崩溃发生时的数据有效性。

参数

WORK
TRANSACTION

可选关键字，没有任何作用。

注意

使用ROLLBACK语句退出事务。

在一个事务外部发出COMMIT不会有问题，但是将产生一个警告信息。

例子

提交当前事务以让所有变更永久化： 

COMMIT;

兼容性

SQL标准只声明了COMMIT和COMMIT WORK两种形式。 否则这条命令与标准完全兼容。
